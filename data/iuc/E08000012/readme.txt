
CDRC 2014 IUC Geodata Pack - Liverpool (E08000012)

+ Abstract:

The 2014 Internet User Classification (IUC) is a UK geodemographic of Internet use and engagement. This CDRC 2014 IUC Geodata Pack provides a mappable version of this classification for the Local Authority District: Liverpool (E08000012)

+ Contents:
	- E08000012_Liverpool (.shp,.prj,.dbf,.shx): Shapefile
	- metadata.xml: Meta Data
	- IUC_User_Guide.pdf: User guide for the 2014 Internet User Classification
	- readme.txt: Information about the CDRC 2015 IUC Geodata

+ Attributes in the shapefile:
	- lsoa_cd: Lower Layer Super Output Area (LSOA) code
	- lsoa_nm: Lower Layer Super Output Area (LSOA) name
	- supgrp_cd: Supergroup code for this LSOA
	- supgrp_nm: Supergroup description for this LSOA
	- grp_cd: Group code for this LSOA
	- grp_nm: Group description for this LSOA
